# 22. Справочник BaseGuid-ов и $type для Directum RX

## Зачем нужен BaseGuid

Каждая сущность в .mtd файле имеет `BaseGuid` — GUID родительского типа, от которого она наследуется. Это определяет:
- Какие свойства и контролы наследуются автоматически
- Какой `$type` используется в метаданных
- К какому модулю принадлежит базовый тип

## $type по типу сущности

| Тип сущности | $type |
|---|---|
| Справочник (DatabookEntry) | `Sungero.Metadata.EntityMetadata, Sungero.Metadata` |
| Документ (любой) | `Sungero.Metadata.DocumentMetadata, Sungero.Content.Shared` |
| Задача | `Sungero.Metadata.TaskMetadata, Sungero.Workflow.Shared` |
| Задание | `Sungero.Metadata.AssignmentMetadata, Sungero.Workflow.Shared` |
| Уведомление | `Sungero.Metadata.NoticeMetadata, Sungero.Workflow.Shared` |
| Отчёт | `Sungero.Metadata.ReportMetadata, Sungero.Reporting.Shared` |
| Модуль | `Sungero.Metadata.ModuleMetadata, Sungero.Metadata` |
| Решение | `Sungero.Metadata.SolutionMetadata, Sungero.Metadata` |
| Дочерняя коллекция | `Sungero.Metadata.EntityMetadata, Sungero.Metadata` |

### Custom child entities (табличные части)

| Тип | BaseGuid |
|---|---|
| Custom ChildEntity | `a3d38bf5-0414-41f6-bb33-a4621d2e5a60` |

Свойства: `IsChildEntity: true` (БЕЗ `IsAutoGenerated`).

**КРИТИЧНО для ВСЕХ child entities (auto и custom):**
1. ПОЛНАЯ структура .mtd (CreationArea, RibbonCard, RibbonCollection) — минимальные заглушки → NullReferenceException!
2. Properties с `IsReferenceToRootEntity: true` (ссылка на родителя)
3. `DomainApi: 2` в Versions
4. Custom child entity: первое свойство **БЕЗ PreviousPropertyGuid** (Status `1dcedc29` НЕ существует в ChildEntity base!)

## Примеры использования

### Создать справочник (наследник DatabookEntry)
```json
{
  "$type": "Sungero.Metadata.EntityMetadata, Sungero.Metadata",
  "NameGuid": "<новый-GUID>",
  "Name": "LeadSource",
  "BaseGuid": "04581d26-0780-4cfd-b3cd-c2cafc5798b0"
}
```

### Создать документ (наследник OfficialDocument)
```json
{
  "$type": "Sungero.Metadata.DocumentMetadata, Sungero.Content.Shared",
  "NameGuid": "<новый-GUID>",
  "Name": "CommercialProposal",
  "BaseGuid": "58cca102-1e97-4f07-b6ac-fd866a8b7cb1"
}
```

### Создать задачу (наследник Task)
```json
{
  "$type": "Sungero.Metadata.TaskMetadata, Sungero.Workflow.Shared",
  "NameGuid": "<новый-GUID>",
  "Name": "ProposalApprovalTask",
  "BaseGuid": "d795d1f6-45c1-4e5e-9677-b53fb7280c7e"
}
```
