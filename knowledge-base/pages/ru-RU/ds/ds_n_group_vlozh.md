# Группы вложений

В Solo и Jazz документы вкладываются в в последнюю из групп вложений, доступных для изменения. Как правило, это группа «Дополнительно».

Основным документом считается первый документ из первой обязательной группы вложений задания. Он подписывается при согласовании и отображается при открытии карточки заданий с настроенным в SungeroAdapter.config режимом отображения «согласование» и «рецензирование» (атрибут viewMode элемента задания ).
