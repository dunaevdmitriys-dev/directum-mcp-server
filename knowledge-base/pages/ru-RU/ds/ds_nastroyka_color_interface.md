# Настройка цветовой темы интерфейса

В Development Studio для индивидуального комфорта каждого сотрудника доступны темы оформления интерфейса – темная и светлая, позволяющие настроить цветовую схему в соответствии с вашими предпочтениями и рабочей средой:

Чтобы изменить тему:

- 1. В правом верхнем углу нажмите на кнопку .
- 2. В выпадающем списке выберите тему:
- Системная . При использовании тема Development Studio соответствует вашей операционной системе. Значение по умолчанию;
- Светлая ;
- Темная .

Совет . Используйте единые визуальные стили для редактора кода и среды разработки. Это создает интуитивно понятную цифровую среду и сократит время на адаптацию при переходе между ними.
