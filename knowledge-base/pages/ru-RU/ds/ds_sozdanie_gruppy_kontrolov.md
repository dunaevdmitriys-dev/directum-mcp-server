# Создание группы контролов

На панели фильтрации можно добавлять контролы, с помощью которых задаются параметры фильтрации. Например, чтобы найти все документы, созданные за месяц, достаточно в контроле Период Дат указать За последние 30 дней . При этом контролы добавляются на панель фильтрации группами.

СОВЕТ. При группировке контролов используйте рекомендации .

Чтобы создать группу контролов:

- 1. Перейдите в редактор:

• типа сущности на вкладку «Панель фильтрации», если настраивается панель фильтрации списка типа сущности;

• модуля на вкладку «Папки» в узел «Панель фильтрации», если настраивается панель фильтрации папки.

- 2. Добавьте группу по кнопке :
- 3. В открывшемся редакторе задайте настройки:
- Имя группы контролов. Должно быть уникальным в рамках панели фильтрации, состоять из букв латинского алфавита и цифр, содержать от 2 до 50 символов и начинаться с буквы.
- Отображаемое имя группы контролов, которое показывается в проводнике. Для локализации имени перейдите к его полю и нажмите на кнопку .
- Тип контрола . Устанавливается при создании группы и недоступен для редактирования.
- Отображать в панели фильтрации . По умолчанию созданная группа контролов отображается в проводнике. При необходимости ее можно скрыть, например при перекрытии. Для этого снимите флажок.
- 4. Добавьте контролы в группу. Для этого в узле «Панель фильтрации» перейдите к ней и нажмите на кнопку . В выпадающем списке выберите тип создаваемого контрола:

• Период дат . Используется для добавления возможности фильтровать по временному промежутку. Например, чтобы получить документы, созданные за последние 7 дней, отфильтруйте их список по дате;

• Набор флажков . Используется для добавления флажков, которые позволяют выбрать один или несколько вариантов из предложенного списка. Например, чтобы найти не зарегистрированные и зарезервированные договоры, на панели фильтрации в группе «Состояние» установите соответствующие флажки;

• Ссылка на сущность . Используется для фильтрации записей по значению конкретного типу сущности. Например, предположим, что у договоров есть свойство Журнал регистрации , при этом возможные значения выбираются из соответствующего справочника. В таком случае с помощью контрола «Ссылка на сущность» можно фильтровать записи в списке по значению свойства Журналу регистрации .
