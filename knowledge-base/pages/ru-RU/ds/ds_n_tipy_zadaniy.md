# Поддержка типов заданий в NOMAD

Поддерживаются все типы заданий со следующими ограничениями:

- отображается только информация с закладки «Свойства» карточки задания. Дополнительные закладки не поддерживаются;
- на форму карточки задания нельзя добавить контрол состояния, метки и ссылки;
- доступны не все типы свойств задания;
- ограничена работа с группами вложений ;
- для отображения действий (не вариантов выполнения) необходима настройка ;
- доступны не все типы диалогов .

Ограничения вариантов выполнения и действий:

- не открываются карточки сущностей, списков, отчетов и т.п., кроме создания новой задачи;
- действия выполняются по умолчанию в онлайн-режиме. Для выполнения их офлайн необходима настройка ;
- поставить утверждающую подпись можно только при согласовании по регламенту. Подписание нужно настроить в конфигурационном файле SungeroAdapter.config .
