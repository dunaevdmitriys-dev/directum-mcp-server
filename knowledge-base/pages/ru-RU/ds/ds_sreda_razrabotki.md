# Среда разработки

Текст раздела в разработке

Среда разработки Development Studio предназначена для модификации цифровой платформы Directum RX под потребности компании. Вы можете:

- расширять функциональность, создавая собственные решения и модули ;
- адаптировать объекты под свои нужды, разрабатывать собственные типы сущностей , используя механизм наследования ;
- изменять стандартные возможности с помощью механизма перекрытия .

Development Studio имеет современный и интуитивно понятный интерфейс. Логичная структура меню и четкая организация элементов позволяют без труда находить необходимые функции, а удобный дизайн делает процесс работы не только эффективным, но и комфортным.

Редактор . У каждого элемента разработки (решения, модуля, типа сущности или отчета) есть свой редактор. В нем на разных вкладках задаются настройки, добавляются функции и события. Чтобы открыть редактор, в дереве решений щелкните левой кнопкой мыши по названию элемента разработки.

Дерево рабочих решений . В дереве отображается список разрабатываемых вами решений. В его узлах отображаются перекрытия элементов базовых решений и созданные модули, типы сущностей и отчеты. Чтобы открыть редактор нужного элемента, например модуля, щелкните левой кнопкой мыши по его названию в дереве решений.

Дерево базовых решений . В дереве отображается список базовых решений, которые доступны только для просмотра.
