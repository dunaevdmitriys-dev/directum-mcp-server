# Отчеты

Отчеты позволяют получить данные из системы и отобразить их в удобном виде во внешних приложениях.

ВАЖНО. В текущей версии работа с отчетами поддерживается только в Directum RX Development Studio. Подробнее о создании отчетов и работе с ними см. в документации Directum RX Development Studio, раздел «Отчеты» .
