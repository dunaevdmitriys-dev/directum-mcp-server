# Возможное содержимое группы вложений

Чтобы разрешить добавлять в группу вложений ссылки только на определенный тип сущности, перейдите к нужной группе и по кнопке создайте ограничение на ее содержимое. В открывшемся редакторе заполните поля:

Имя ограничения на содержимое группы. Задается автоматически при указании типа сущности в поле Тип . Недоступно для изменения.

Включить ограничение на тип сущности . Установите флажок, если группа получена при перекрытии задачи и нужно снять все ограничения на ее вложения. Флажок доступен для типов сущностей, которые добавлены в группу вложений на рабочем слое.

Тип сущности, ссылки на которую пользователи могут добавлять в группу.

Количество вложений выбранного типа, которое может быть в группе. Если значение не указано, в группу можно добавлять любое количество. Например, чтобы в группу «На согласование» можно было добавлять только один согласуемый документ, укажите значение 1 .
