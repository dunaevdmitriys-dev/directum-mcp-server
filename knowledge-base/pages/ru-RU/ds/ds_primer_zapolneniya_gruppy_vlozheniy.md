# Пример заполнения группы вложений

Предположим, в задаче на согласование заявки на закупку нужно выделить группы «Заявка» и «Приложения». При этом в группу «Приложения» должны автоматически добавляться приложения к документу из группы «Заявка». Для этого:

- 1. На вкладке «Группы вложений» выберите группу вложений AddendaGroup (Приложения):
- 2. Задайте свойства группы вложений.
- 3. В обработчике события «Заполнение вложений» добавьте код, чтобы в группу AddendaGroup (Приложения) добавлялись приложения к документу из группы PurchaseApplication (Заявка).
- СОВЕТ. В нашем примере для получения приложений можно использовать функцию базового решения GetAddenda.

```csharp
public override void AddendaGroupPopulating(Sungero.Workflow.Interfaces.AttachmentGroupPopulatingEventArgs e)
{
/// С помощью обработчика GetAddenda получить
/// приложения вложений из группы ForApprovalGroup (На согласование)
/// и заполнить ими группу Addenda (Приложения).
e.PopulateFrom(_obj.ForApprovalGroup, GetAddenda);
}
```

В результате в группу «Приложения» автоматически добавляются и обновляются приложения к документу из группы «Заявка».
