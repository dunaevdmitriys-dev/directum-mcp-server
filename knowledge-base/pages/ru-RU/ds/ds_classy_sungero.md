# Классы Sungero

## Классы Sungero

Разрешенные классы | Запрещенные классы
Классы из пространства имен всех прикладных модулей: Sungero.Company , Sungero.Commons и т.д. | Классы всех остальных пространств имен Sungero.*
Классы из пространства имен всех предметных модулей: Sungero.Content и Sungero.Workflow
Классы из системных пространств имен. Разрешается использовать только Sungero.Core и Sungero.CoreEntities

ВАЖНО. Запрещенные классы использовать нельзя, так как при обновлении системы не гарантируется совместимость кода.
