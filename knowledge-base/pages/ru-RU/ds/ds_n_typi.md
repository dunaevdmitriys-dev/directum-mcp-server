# Типы

## Тип задачи

Тип задачи (см. наследников «Sungero.Workflow.ITask»). Например, «Sungero.Docflow.IApprovalTask».

## Тип задания

Тип задания (см. наследников «Sungero.Workflow.IAssignment»). Например, «Sungero.Docflow.IFreeApprovalAssignment».

## Тип документа

Тип документа (см. наследников «Sungero.Content.IElectronicDocument»). Например, «IIncomingDocumentBase».

## Тип подписи

Тип подписи (см. «Sungero.Core.SignatureType»). Например, «Approval».
