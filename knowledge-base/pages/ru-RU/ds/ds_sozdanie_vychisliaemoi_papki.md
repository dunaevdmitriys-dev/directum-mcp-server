# Пример создания вычисляемой папки

Предположим, необходимо создать папку «Заявки на согласовании». Папка должна содержать заявки, отправленные на согласование, и находиться в модуле «Закупки». Для этого:

- 1. В редакторе модуля «Закупки» на вкладке «Папки» нажмите на кнопку .
- 2. На вкладке «Основное» в группах «Именование» и «Параметры» заполните поля:
- Имя . Укажите тип сущности PurchasesOnApproval .
- Отображаемое имя . Укажите значение Заявки на согласовании .
- Область отображения . Укажите значение Модуль .
- Основное содержимое . Укажите значение Sungero.Purchase.PurchaseRequest .
- Также добавьте иконку и проверьте, что установлен флажок Отображать папку .
- Подробнее о параметрах см. в разделе «Редактор папки» .
- 3. Чтобы в папку попадали только заявки на согласовании, перейдите на вкладку «Обработчики» в группу «События» к обработчику события Получение данных и нажмите на кнопку .

```csharp
/// <summary>
/// Выбор из списка всех заявок на согласование.
/// </summary>
/// <param name="query">Список заявок для фильтрации.</param>
/// <returns>Отфильтрованный список заявок.</returns>
}public virtual IQueryable<Sungero.Purchases.IPurchaseRequest> PurchasesOnApprovalDataQuery(IQueryable<Sungero.Purchases.IPurchaseRequest> query)
{
// Фильтр по статусу документа.
return query.Where(x => x. InternalApprovalState == InternalApprovalState.OnApproval);
}
```

- 4. В редакторе модуля «Закупки» на вкладке «Главная» укажите зависимость папки от модуля Sungero.Puchases, так как на вкладке «Папки» в параметре *Основное содержимое указано значение Sungero.Puchases.PuchasesRequest .
- Важно. Если зависимости не указаны, то при публикации могут возникнуть ошибки.
- 5. При необходимости настройте внешний вид папки по умолчанию и панель фильтрации .
- 6. Чтобы папка появилась в проводнике, при инициализации модуля выдайте на нее права доступа. Для этого в редакторе модуля на вкладке «Обработчики» в группе «События» рядом со ссылкой Инициализация нажмите на кнопку .
- 7. В открывшемся редакторе кода:

• измените имя Function() на GrantRightsOnFolder() ;

• в функции GrantRightsOnFolder() напишите код для выдачи прав на вычисляемую папку:

```csharp
/// <summary>
/// Функция инициализации для выдачи прав на вычисляемую папку.
/// </summary>
public static void GrantRightsOnFolder()
{
var allUsers = Roles.AllUsers;
Sungero.Purchases.SpecialFolders.PurchasesOnApproval.AccessRights.Grant(allUsers, DefaultAccessRightsTypes.Read);
Sungero.Purchases.SpecialFolders.PurchasesOnApproval.AccessRights.Save();
InitializationLogger.Debug("Выданы права на вычисляемую папку 'Заявки на согласовании'");
}
```

- ПРИМЕЧАНИЕ. Для обращения к вычисляемым папкам в коде используется формат: <Код компании>.<Имя модуля>.SpecialFolders.<Имя вычисляемой папки>. Например, Sungero.Purchases.SpecialFolders.PurchasesOnApproval.

• добавьте вызов созданной функции в обработчике события Initializing() :

```csharp
public partial class ModuleInitializer
{
/// <summary>
/// Обработчик события инициализации.
/// </summary>
public override void Initializing(Sungero.Domain.ModuleInitializingEventArgs e)
{
/// Вызов функции инициализации.
GrantRightsOnFolder();
}
}
```

В результате после публикации решения и инициализации модуля «Закупки» в нем отображается папка «Заявки на согласовании». В ней содержатся отправленные на согласование заявки на закупки.
