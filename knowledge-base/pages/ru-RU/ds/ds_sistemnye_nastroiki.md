# Общесистемные настройки

Общесистемные настройки расширяют возможности работы с Development Studio и не являются обязательными. Настройки задаются в среде разработки. Также может потребоваться установка и настройка дополнительных компонентов.

После установки среды разработки выполните необходимые вам настройки:

- 1. Для совместной работы в Git нескольких разработчиков установите и настройте систему управления репозиториями, например Gitea.
- 2. Настройте цветовую тему интерфейса.
