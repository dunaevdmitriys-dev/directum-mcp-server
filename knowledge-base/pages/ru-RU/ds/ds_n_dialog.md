# Диалоги

## Directum Solo

Поддерживаемые типы диалогов:

- сообщение об ошибке, предупреждение – ShowMessage ;
- всплывающее сообщение – NotifyMessage ;
- диалог с выбором варианта – CreateTaskDialog ;
- диалог с запросом параметров – CreateInputDialog .

Сообщения об ошибке и предупреждения отображаются вместе с результатами проверки задания.

Поддерживаемые элементы в диалоге запроса параметров:

- флажок;
- дата;
- вещественное число;
- целое число;
- строка;
- многострочный текст;
- строка ввода пароля;
- выпадающий список.
