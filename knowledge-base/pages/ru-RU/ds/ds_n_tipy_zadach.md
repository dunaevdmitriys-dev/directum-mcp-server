# Поддержка типов задач в NOMAD

В мобильных приложениях поддерживается создание:

- простых задач;
- задач на исполнение поручений;
- задач, отмеченных в конфигурационном файле SungeroAdapter.config атрибутом allowStart="true" . В текущей версии приложения это задачи на согласование по процессу. регламенту и свободное согласование.

В силу ограниченной поддержки произвольных типов задач остальные типы считаются неподдерживаемыми.

Особенности работы задач в приложении:

- 1. Не поддерживается переопределение свойств задач и их событий.
- 2. Не поддерживается работа с диалогами.
- 3. Не поддерживается логика «Ведущий-ведомый» в свойствах.
- 4. При старте задач автоматически выдаются права:

• для простых задач – на вложенные документы;

• для остальных типов задач – на документы из групп, указанных в секциях autoRightsAttachmentGroup в конфигурационном файле SungeroAdapter.config.

- При этом учитываются параметры конфигурационного файла Nomad.config, определяющие права доступа: autoRightsOnAttachments , autoRightsForPerformers , autoRightsForObservers .

Для быстрого поиска связей между поручением и совещанием, в рамках которого отправлено поручение, используется реквизит записи поручения Meeting . При этом в реквизите указывается ИД совещания.
