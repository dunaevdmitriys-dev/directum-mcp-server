# If – дополнительное действие запрашиваемого параметра

Дополнительное атомарное действие запрашиваемого параметра.

value . Значение параметра.

action . Действия.

Пример

```csharp
<if value="true">
<action name="SetLifeCycle" target="DocumentGroup" state="Obsolete" />
</if>
```

Дополнительное действие установки стадии жизненного цикла документов при значении запрашиваемого параметра true .
