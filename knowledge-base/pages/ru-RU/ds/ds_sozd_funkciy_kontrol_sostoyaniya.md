# Создание функций контрола состояния

Контрол состояния предназначен для отображения информации, которая может понадобиться при работе с сущностью. Используйте контрол, например, чтобы в карточке задачи показывать данные о ее этапах. Информация в контроле состояния отображается в виде блоков.

Предположим, необходимо, чтобы в карточке заявки на закупку отображалась краткая информация о задачах, созданных по документу. Для этого:

- 1. В редакторе типа документа PurchaseRequest модуля Purchases в группе «Функции» рядом со ссылкой Серверные нажмите на кнопку .
- 2. В открывшемся редакторе измените имя созданной функции, добавьте комментарий с описанием и добавьте методы для создания блоков с информацией.
- СОВЕТ. При создании функции рекомендуется использовать атрибут LocalizedFunction . В таком случае в настройках представления будет отображаться локализованное название функции, что упростит ее поиск.
- 3. Сохраните и опубликуйте изменения.
- 4. В проводнике системы настройте карточку типа сущности PurchaseRequest :

• создайте вкладку;

• добавьте на нее контрол состояния. В поле Функция укажите функцию Создать контрол состояния заявки на закупку .
