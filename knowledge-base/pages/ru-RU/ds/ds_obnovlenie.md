# Обновление

Для обновления среды разработки Development Studio используется кроссплатформенный инструмент Directum Launcher .

Чтобы обновить среду разработки:

- 1. Выполните подготовительные действия .
- 2. Обновите среду разработки на нужную версию.
- 3. Ознакомьтесь с изменениями базового решения Directum RX .
