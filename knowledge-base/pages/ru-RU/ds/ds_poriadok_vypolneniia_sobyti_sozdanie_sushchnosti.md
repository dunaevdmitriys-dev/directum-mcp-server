# Создание сущности

Создание сущности вызывается:

- в клиентском приложении. Например, при нажатии на кнопку Создать или Создать копию в проводнике системы;
- на сервере. Например, при вызове метода Create() в серверных функциях и обработчиках событий.

Порядок создания сущности копированием:

Важно. При создании сущности, например по кнопке Создать , выполняется только обработчик события Создание .
