# Структуры

Структуры – объекты, которые состоят из нескольких логически связанных между собой полей разного типа. Например, в структуре можно хранить персональную информацию о сотруднике: ФИО, дату рождения, паспортные данные.

Полями в структурах могут быть:

• простые типы:

short, short? | Guid, Guid? | double, double?
int, int? | bool, bool? | Uri
long, long? | decimal, decimal? | string
char, char? | DateTime, DateTime?

- перечисления: Enumeration, Enumeration?;
- ссылки на типы сущностей: IUser, ICompany и т.п.;
- интерфейсы структур и их списки;
- списки простых типов или типов сущностей: List<int>, List<ICompany> и т.п.

## Создание структуры

```csharp
/// <summary>
/// Замещение.
/// </summary>
partial class Substitution
{
/// <summary>
/// Замещающий.
/// </summary>
public IUser User { get; set; }

/// <summary>
/// Замещаемый.
/// </summary>
public IUser SubstitutedUser { get; set; }
}
```

## Структура без атрибута Public

```csharp
/// <summary>
/// Исполнительская дисциплина подразделения.
/// </summary>
partial class DepartmentDiscipline
{
/// <summary>
/// Процент исполнительской дисциплины сотрудников.
/// </summary>
public int? Discipline { get; set; }

/// <summary>
/// Подразделение.
/// </summary>
public Sungero.Company.IDepartment Department { get; set; }
}
```

## Структура с атрибутом Public

```csharp
/// <summary>
/// Отпуск.
/// </summary>
[Public]
partial class Vacation
{
/// <summary>
/// Дата начала отпуска.
/// </summary>
public DateTime StartVacation { get; set; }

/// <summary>
/// Дата завершения отпуска.
/// </summary>
public DateTime EndVacation { get; set; }

/// <summary>
/// Отпускник.
/// </summary>
public IUser Vacationer { get; set; }
}
```

Формат создания экземпляра структуры
