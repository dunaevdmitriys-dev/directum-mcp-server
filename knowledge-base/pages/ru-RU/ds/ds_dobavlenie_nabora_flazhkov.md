# Контрол типа «Набор флажков»

Контрол используется для добавления флажков, которые позволяют выбрать один или несколько вариантов из предложенного списка. Например, чтобы найти не зарегистрированные и зарезервированные договоры, на панели фильтрации в группе «Состояние» установите соответствующие флажки. Чтобы создать контрол типа «Набор флажков»:

- 1. В узле «Панель фильтрации» перейдите к нужной группе нажмите на кнопку . В выпадающем списке выберите Набор флажков :
- 2. В открывшемся редакторе задайте параметры:
- Имя контрола. Должно быть уникальным в рамках группы, состоять из букв латинского алфавита и цифр, содержать от 2 до 50 символов и начинаться с буквы.
- Отображаемое имя контрола, которое используется в проводнике. Для локализации имени перейдите к его полю и нажмите на кнопку .
- ПРИМЕЧАНИЕ. Чтобы в качестве отображаемого имени набора флажков использовалось название группы, оставьте поле Отображаемое имя незаполненным.
- Тип контрола . Устанавливается при создании контрола и недоступен для редактирования.
- Отображать в панели фильтрации . По умолчанию флажок установлен. Если необходимо скрыть контрол, например при перекрытии, снимите флажок.
- Разрешить множественное выделение флажков . По умолчанию в наборе флажков можно выбирать только один. Если необходимо разрешить пользователю указывать несколько значений набора одновременно, установите флажок.
- Контрол «флажок» для управления доступностью . Если нужно, чтобы в проводнике контрол был доступен только при установке выбранного флажка другого контрола «Набор флажков», задайте в поле нужный флажок. В выпадающем списке для выбора доступны флажки из всех наборов, которые добавлены на панель фильтрации, кроме флажков создаваемого контрола.
- Если поле не заполнено, контрол будет доступен всегда.
- 3. Создайте флажки контрола. Для этого перейдите к нему и последовательно добавьте флажки по кнопке .
- 4. В редакторе каждого флажка задайте параметры:
- Имя флажка. Должно быть уникальным в рамках группы, состоять из букв латинского алфавита и цифр, содержать от 2 до 50 символов и начинаться с буквы.
- Отображаемое имя флажка, которое используется в проводнике. Для локализации имени перейдите к его полю и нажмите на кнопку .
- Тип контрола . Устанавливается при создании флажка и недоступен для редактирования.
- Отображать в панели фильтрации . По умолчанию созданный флажок отображается в проводнике. При необходимости его можно скрыть, например при перекрытии. Для этого снимите флажок.
- Выделен по умолчанию . Установите, чтобы в проводнике флажок выбирался по умолчанию. Рекомендуется использовать для значений, по которым фильтрация выполняется чаще всего.
- 5. Перейдите в редактор панели фильтрации и добавьте обработчик события «Фильтрация» . В нем задайте логику использования созданного контрола.
