# Администрирование Development Studio

Текст раздела в разработке

В руководстве содержатся описания установки, настройки и сопровождения среды разработки Development Studio. Эти действия выполняет разработчик на своем компьютере:

- 1. Ознакомьтесь с требованиями к аппаратному и программному обеспечению.
- 2. Установите или обновите Development Studio.
- 3. Настройте конфигурацию среды разработки, если после ее установки изменились какие-либо данные, например код компании. Конфигурирование настроек выполняется с помощью инструмента Directum Launcher.
- 4. Установите сторонний редактор кода и настройте интеграцию среды разработки с ним.
- 5. При необходимости выполните дополнительную настройку Development Studio. Например, для совместной работы нескольких разработчиков настройте работу с системой контроля версий Git.

Примечание. Предыдущая среда разработки Directum RX Development Studio оставлена для совместимости и решения задач, которые первое время еще будут недоступны в Development Studio. Ознакомьтесь с настройкой параллельной работы в этих двух средах.
