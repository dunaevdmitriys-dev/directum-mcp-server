# param – свойство

code . Имя свойства.

Пример :

```csharp
<result name="Accept">
<params>
<param code="NewDeadline" />
<param code="ScheduledDate" />
</params>
</result>
```

Для варианта выполнения задаются отображаемые свойства - новый срок и текущий срок.
