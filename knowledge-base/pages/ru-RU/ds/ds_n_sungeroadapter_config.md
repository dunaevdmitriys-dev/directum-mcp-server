# Настройка задач и заданий в файле SungeroAdapter.config

В файле SungeroAdapter.config содержатся настройки, которые обеспечивают совместную работу Directum RX с сервисом NOMAD, а также мобильными приложениями Solo и Jazz:

- task – описание задачи;
- job – описание задания или уведомления;
- action – описание действия;
- result – вариант выполнения задания с дополнительной логикой;
- autoRightsAttachmentGroup – группа вложений, на документы из которой автоматически назначаются права;
- ask – запрашиваемые параметры;
- If – дополнительное действие запрашиваемого параметра;
- condition – условия, наложенные на действие;
- param – свойство;
- taskDialog – диалог с выбором варианта для автоматической обработки;
- button – кнопка диалога;
- Типы ;
- Бизнес-процессы ;
- Документы ;
- Флаги .

Расположение файла по умолчанию см. в разделе «Сопровождение мобильных решений» ( Linux , Windows ).

ВАЖНО. Новые настройки применяются к задачам и заданиям, которые созданы после сохранения изменений в конфигурационном файле SungeroAdapter.config.

Примеры по адаптации сервиса NOMAD для собственных доработок см. в разделе «Примеры доработок в файле SungeroAdapter.config» .
