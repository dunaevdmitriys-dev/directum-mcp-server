# autoRightsAttachmentGroup – группа вложений, на документы из которой автоматически назначаются права

target . Код группы вложений. На документы из заданной группы автоматически назначаются права при старте задачи. Назначение прав на документы из групп «Документ» и «Приложения» обрабатывается автоматически и не требует дополнительной настройки.

Пример

```csharp
<autoRightsAttachmentGroup target="OtherGroup" />
```

Указан код группы вложений для автоматического назначения прав.
