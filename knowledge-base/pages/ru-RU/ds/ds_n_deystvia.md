# Действия

В мобильных приложениях в карточке задания отображаются все доступные варианты выполнения. Если нужно отобразить определенное действие, укажите его в конфигурационном файле SungeroAdapter.config.

Пример:

```csharp
<job type="Sungero.Docflow.IApprovalAssignment">
<result name="AddApprover"/>
</job>
```

В задании на согласование вместе с вариантами выполнения будет отображено действие «Добавить согласующего».
