# Настройка панели фильтрации

Панель фильтрации (фильтр) используется для поиска записей по определенным критериям в вычисляемых папках или списках типов сущностей . Например, с помощью нее в списке входящих документов можно отфильтровать записи по журналу регистрации, виду документа или по дате создания.

Панель фильтрации рекомендуется добавлять, когда:

- требуются выборки записей, которые сотрудник не может настроить через интерфейс системы, например просроченные задания или фильтрация по типам сущностей;
- необходимо использовать несколько представлений одного списка для разных ролей пользователей и сценариев работы;
- нужна фильтрация по свойствам, которые не вынесены в столбцы списка;
- требуется фильтрация по временному периоду, который сложно настроить с помощью фильтрации дат в столбце.

Панель фильтрации состоит из групп, каждая из которых содержит контролы . Они могут быть следующих типов:

- набор флажков. Контрол используется для выбора одного или нескольких значений из фиксированного набора возможных значений. Он может отображаться в виде набора флажков или переключателя;
- ссылка на сущность. Контрол используется для выбора сущности заданного типа. Он может отображаться в виде выпадающего списка или формы-списка;
- период дат. Контрол используется для выбора интервала времени.

СОВЕТ. При выборе типа контрола используйте рекомендации .

## Настройка панели фильтрации

- 1. Если настраивается панель фильтрации:

• списка, то в редакторе типа сущности перейдите на вкладку «Панель фильтрации»;

• папки, то в редакторе модуля перейдите на вкладку «Папки» в узел «Панель фильтрации».

- 2. По умолчанию панель фильтрации отображается в проводнике. Чтобы ее скрыть, перейдите на вкладку «Основное» и снимите флажок Включить панель фильтрации .
- 3. Перейдите на вкладку «Обработчики» и добавьте обработчики событий. Например, чтобы предотвратить отправку запросов, которые возвращают тысячи записей и замедляют работу системы, с помощью обработчика события «Проверка фильтра» добавьте для панели фильтрации проверку ее параметров.
- 4. Добавьте группы контролов , чтобы с помощью них фильтровать данные.
- 5. В проводнике настройте представление панели фильтрации.
