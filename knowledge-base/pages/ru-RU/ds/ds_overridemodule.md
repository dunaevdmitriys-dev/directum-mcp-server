# Шаг 2. Перекрытие модуля

- 1. В дереве решения, созданного на шаге 1 , рядом с узлом «Перекрытые элементы» нажмите на , затем выберите пункт Модуль .
- 2. В открывшемся окне в выпадающем списке выберите модуль «Договоры» ( Sungero.ContractsUI ) из базового решения Directum RX:
- 3. Нажмите на кнопку Перекрыть . В результате модуль появляется в дереве решения в пункте Перекрытые элементы , и открывается его редактор.
- 4. Перейдите в редактор вашего решения и добавьте зависимость от модуля Sungero.ContractsUI.
- 5. Чтобы получить список договоров с резидентами, в редакторе перекрытого модуля «Договоры» (ContractUI) на вкладке «Главная» в группе «Функции» рядом со ссылкой Серверные нажмите на кнопку .
- 6. В открывшемся редакторе измените имя созданной функции, добавьте комментарий с описанием и вставьте код:

```csharp
/// <summary>
/// Получить договоры с резидентами.
/// </summary>
/// <returns>Договоры с резидентами.</returns>
[Remote]
public IQueryable<Purchases.IContract> GetContractsWithResidents()
{
return Purchases.Contracts.GetAll(t => t.Resident.Value);
}
```

- 7. Чтобы в системе можно было показать список договоров с резидентами, в редакторе перекрытого модуля «Договоры» (ContractUI) на вкладке «Главная» в группе «Функции» рядом со ссылкой Клиентские нажмите на кнопку .
- 8. В открывшемся редакторе измените имя созданной функции, добавьте комментарий с описанием и вставьте код:

```csharp
/// <summary>
/// Показать список договоров с резидентами.
/// </summary>
public virtual void ShowContractsWithResidents()
{
Functions.Module.Remote.GetContractsWithResidents().Show();
}
```

9. Чтобы в системе можно было показать список договоров с резидентами, в редакторе перекрытого модуля «Договоры» (ContractUI) на вкладке «Главная» в группе «Функции» рядом со ссылкой Клиентские нажмите на кнопку .

10. В открывшемся редакторе измените имя созданной функции, добавьте комментарий с описанием и вставьте код:

```csharp
/// <summary>
/// Показать список договоров с резидентами.
/// </summary>
[LocalizeFunction("Договоры с резидентами", "Возвращает список договоров с резидентами.")]
public virtual void ShowContractsWithResidents()
{
Functions.Module.Remote.GetContractsWithResidents().Show();
}
```

- СОВЕТ. При создании функции рекомендуется использовать атрибут LocalizedFunction . В таком случае в настройках представления модуля будет отображаться локализованное название функции, что упростит ее поиск.

11. Опубликуйте разработку.

- 10. В веб-клиенте создайте представление модуля «Договоры». На вкладке «Обложка» добавьте действие Договоры с резидентами . В его свойствах укажите тип действия Функция и функцию Договоры с резидентами .
- 11. Перейдите на обложку модуля «Договоры» и нажмите на ссылку Договоры с резидентами :

Открывается список с созданными договорами с резидентами:
