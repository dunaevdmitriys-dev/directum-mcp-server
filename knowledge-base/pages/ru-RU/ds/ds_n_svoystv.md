# Свойства

В Solo и Jazz поддерживаются типы свойств: дата, перечисление, ссылка, логическое, вещественное, целое, строка, текст.

Не поддерживается логика «Ведущий-ведомый» в свойствах.

В карточке задания отображается вся информация с закладки «Свойства». Свойства из группы Header (содержит контролы, в которых отображается общая информация по заданию: тема, инициатор, исполнители и т.п.) отображаются только для чтения. Свойства из группы Footer (содержит контролы, в которые пользователи вводят информацию о результатах выполнения задания) отображаются для чтения и редактирования.
